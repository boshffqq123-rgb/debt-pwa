# نظام إدارة الديون — تطبيق PWA قابل للتثبيت

## محتويات الحزمة
- `index.html` — التطبيق نفسه
- `manifest.json` — بيانات التطبيق (الاسم، الأيقونة، وضع العرض)
- `sw.js` — Service Worker يخزّن التطبيق كاملاً للعمل بدون إنترنت
- `icons/icon-192.png` و `icons/icon-512.png` — أيقونة التطبيق

قاعدة البيانات (IndexedDB) موجودة أصلاً جوه index.html ولم تتغير — كل بياناتك هتفضل محفوظة على الجهاز نفسه.

---

## الخطوة 1: رفع الملفات أونلاين (شرط أساسي للتثبيت)

متصفح الجوال (Chrome) مايقبلش تثبيت PWA أو تسجيل Service Worker إلا لو الموقع شغال على **HTTPS**. أسهل طريقة مجانية:

### باستخدام GitHub Pages (مجاني، بدون خبرة برمجة)
1. اعمل حساب على github.com لو مالكش.
2. اعمل Repository جديد (اسمه مثلاً `debt-app`) واختار Public.
3. ارفع الملفات الأربعة (`index.html`, `manifest.json`, `sw.js`, ومجلد `icons`) بالكامل داخل الـ Repository (فيه زرار Add file → Upload files).
4. من إعدادات الـ Repository: Settings → Pages → Branch: main → Save.
5. بعد دقيقة، هتلاقي رابط زي: `https://username.github.io/debt-app/`

بدائل سريعة برضو: Netlify Drop (netlify.com/drop — تسحب المجلد وتفلته وخلاص) أو Vercel.

---

## الخطوة 2: تثبيت التطبيق على جالكسي (زي أي تطبيق تاني)
1. افتح الرابط في متصفح **Chrome** على الجوال.
2. هيظهر تلقائي خيار "إضافة إلى الشاشة الرئيسية" أو زرار "📲 تثبيت التطبيق" اللي جوه الصفحة نفسها.
3. دوس تثبيت — هتلاقي أيقونة التطبيق ظهرت في قائمة التطبيقات بتاعتك بنفس شكل باقي التطبيقات، وبتفتح في نافذة مستقلة من غير شريط المتصفح.
4. من هنا التطبيق شغال بالكامل offline، وبياناتك محفوظة محلياً على الجهاز.

---

## الخطوة 3 (اختياري): تحويله لملف APK حقيقي
لو عايز ملف APK تقدر تنصبه يدوي أو تنزله على جوجل بلاي:

1. روح لموقع **PWABuilder**: https://www.pwabuilder.com
2. حط رابط الـ GitHub Pages بتاعك (من الخطوة 1) في المكان المخصص واضغط Start.
3. الموقع هيفحص التطبيق ويأكد إن الـ manifest والـ service worker شغالين تمام.
4. اختار "Android" من الخيارات، وحمّل ملف الـ APK الجاهز.
5. انقل ملف الـ APK على جوالك ونصّبه (لازم تفعّل "السماح بالتثبيت من مصادر غير معروفة" من إعدادات الأمان أول مرة).

ملحوظة: قاعدة البيانات (IndexedDB) هتفضل شغالة بالظبط زي ما هي جوه الـ APK كمان، لأنه أساساً بيغلف نفس صفحة الويب.

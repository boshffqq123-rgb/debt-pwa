# دفتر الحسابات — نظام إدارة الديون (النسخة 2)

نظام PWA يعمل بالكامل دون إنترنت، مبني على IndexedDB، لإدارة ديون عملائك من جهاز واحد.

## التشغيل
افتح `index.html` مباشرة في المتصفح، أو ثبّته كتطبيق (PWA) عبر زر "📲 تثبيت التطبيق" الذي يظهر أعلى الصفحة. يفضّل تشغيله عبر خادم محلي بسيط (وليس فتح الملف مباشرة) حتى يعمل Service Worker والتثبيت بشكل كامل؛ مثال:

```
npx serve .
```

## ماذا تغيّر عن النسخة السابقة؟
- **بنية بيانات صحيحة**: `Customers` (عملاء) + `Transactions` (عمليات دين/سداد) بدل السجل المسطّح القديم. **بياناتك القديمة تُرحَّل تلقائيًا** عند أول تشغيل (تظهر رسالة تأكيد)، ولا حاجة لإعادة إدخال شيء.
- **كشف حساب لكل عميل** بتصميم إيصال، مع الرصيد وكل العمليات وطباعة/PDF/Excel خاص بالعميل.
- **لوحة تحكم** بإحصاءات فورية وأعلى المدينين وآخر العمليات.
- **نسخ احتياطي واستعادة حقيقيان** (تصدير/استيراد JSON كامل، مع خيار دمج أو استبدال).
- **تقارير** بفلاتر تاريخ/عميل/نوع، مع تصدير Excel حقيقي (`.xlsx` عبر مكتبة SheetJS المرفقة محليًا — لا حاجة لإنترنت) و CSV و طباعة/PDF.
- **بحث فوري** عبر العملاء والعمليات معًا.
- **تنبيهات** لتجاوز سقف الدين والتأخر عن السداد.
- **واجهة احترافية** مع وضع ليلي، وتصميم متجاوب للجوال.
- **حماية PIN** اختيارية مع قفل تلقائي بعد 5 دقائق خمول.
- **سجل نشاط (Audit Log)** يوثّق كل إضافة/تعديل/حذف.
- **أرشفة وتثبيت العملاء** من صفحة كشف الحساب.

## بنية الملفات
```
index.html        الهيكل العام والواجهات
css/style.css      التصميم
js/db.js           طبقة قاعدة البيانات + الهجرة التلقائية + النسخ الاحتياطي
js/app.js          كل منطق الواجهة والتنقل
js/lib/xlsx.full.min.js   مكتبة تصدير Excel الحقيقية (مرفقة محليًا، بدون إنترنت)
sw.js, manifest.json      إعدادات PWA
```

## ملاحظة أمان
رمز PIN يُحفظ كـ hash (SHA-256) وليس نصًا صريحًا. تشفير النسخة الاحتياطية نفسها غير مفعّل حاليًا (الملف المصدَّر JSON عادي)، فاحتفظ بنسخك الاحتياطية في مكان موثوق.
